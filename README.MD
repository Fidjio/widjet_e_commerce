Приложение находится в разработке. Конечные цели будут описаны при их достижении.
E-commerce — электронная торговля, или электронная коммерция. Виджет 
является ядром для интернет-магазина. В дальнейшем для этого ядра возможно 
будет реализовать любой интерфейс — от сайта до телеграм-бота.


Виды обработки информации представлены ниже в разделе **"Реализованные функции"**.

## Реализованные функции

### Модуль product.py
1. ***Product()*** - класс, который формирует объект с именем продукта,
описанием продукта, его ценой и количеством продуктов. Вызов класса:\
`prod1 = Product(name, description, price, quantity)` \


### Модуль category.py
1. ***Category()*** - класс, который формирует объект с именем категории,
описанием категории и списком продуктов. Так же в классе доступны атрибуты 
count_category (считает количество объектов) и count_products
(считает количество продуктов в объектах). Вызов класса:\
`cat1 = Category(name, description, list_product)` \



### Модуль utils.py
1. ***read_json*** - открывает json-файл и возвращает словарь. Вызов функции:\
`data = read_json(path_to_file)` -> dict

2. ***create_object_for_json*** - принимает список словарей и 
формирует список с объектами классов Product и Category. Вызов функции:\
`list_objects = create_object_for_json(list_dict_data)` -> list[obj1, obj2, ...]
